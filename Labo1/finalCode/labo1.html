Notes:

  • Les inputs en décimal supportent le format texte! Essayez d'y placer Infinity, NaN ou -0!
  • -0 s'affiche comme étant 0 mais réagis bien tel un -0. Voir le cas "1/(-0) = -Infinity" dans demoFile.
  • 1 ET -1 sont utilisables pour set le signe négatif... pour les étourdis. :-)
  • Dans conversion bin>>dec, pas besoin d'écrire les chiffres en entier. Un exposant de 11 devient 110000000. De plus ne vous inquiétez pas, dépasser le nb de bits ne fait rien (de mal)!
